# Security Thanks

Elio thanks security researchers who report vulnerabilities through the
coordinated disclosure process described in [SECURITY.md](SECURITY.md).

No public security acknowledgments have been recorded yet.

## Recognition Policy

- Reporters are credited only after the vulnerability is validated and fixed.
- Reporters may request anonymity.
- Reports that do not affect Elio, duplicate an existing report, or are
  submitted through public issues instead of the private security process may
  be omitted from this list.
